Class: Widget

Widget

Widgets allow you to add HTML elements on top of the game canvas without blocking input to the game. Each widget is a free-floating div element that can have any HTML content added to it.

new Widget(html, x, y, width, height)

Creates a new instance of Widget.
Parameters:
Name Type Description
html string Initial HTML content for the widget.
x number Initial x position of the widget.
y number Initial y position of the widget.
width number Initial width of the widget.
height number Initial height of the widget.
Properties:
Name Type Description
x number Sets/gets the widget's x position.
y number Sets/gets the widget's y position.
top number Sets/gets the widget's position so that its top edge is at the given position.
bottom number Sets/gets the widget's position so that its bottom edge is at the given position.
left number Sets/gets the widget's position so that its left edge is at the given position.
right number Sets/gets the widget's position so that its right edge is at the given position.
width number Sets/gets the widget's width.
height number Sets/gets the widget's height.
html string Sets/gets the HTML code for the widget's content div.
content div The content div of the widget. This has all of the properties any HTML div would have.
style div style The style object of the content div of the widget. This has all of the properties any HTML style object would have.
