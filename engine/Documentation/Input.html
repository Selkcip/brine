Class: Input

Input

The Input class wraps keyboard and mouse input into one object making it easier to listen for key and mouse actions.

new Input()

Creates a new instance of Input.
Properties:
Name Type Description
mouse Vector The screenspace mouse coordinates relative to the top left corner of the game canvas.
shift bool Is the shift key pressed.
lBtn bool Is the left mouse button pressed.
mBtn bool Is the middle mouse button pressed.
rBtn bool Is the right mouse button pressed.
