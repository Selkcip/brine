FileManager

Singleton that manages file loading. By default there is a global reference called Files.

new FileManager()

Creates a new instance of FileManager or returns the singleton instance.
Properties:
Name Type Description
instance FileManager The singlton instance.
files Array<Object{text,xml}> Array of objects. Each object has a plain text (.text) version of the file as well as an XML (.xml) version.
