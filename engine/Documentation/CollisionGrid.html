Class: CollisionGrid

CollisionGrid

CollisionGrid is used to perform grid based collision detection. This was originally for an isometric game so it's set in the XZ plane, but it's just a 2D grid so it can be used for collisions in screen space as well.

new CollisionGrid(x, z, width, depth, cols, rows)

Creates an instance of CollisionGrid.
Parameters:
Name Type Description
x number X position of the grid in world space.
z number Z (Y) position of the grid in world space.
width number Width of the grid.
depth number Depth of the grid.
cols number Number of horizontal cells.
rows number Number of vertical cells.
